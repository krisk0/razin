*.py                  .py in root directory: Python script, usually benchmark
                       or test, usually requires flint_sage

                        sage. Installs as flint_sage. Tests in separate files test* 

flint.binding/*.README  kinda documentation with installation instruction and
                         FAQ

                         to uname. Re-running the program gives slightly different
                         result.
